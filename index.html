  • Authentification (Authentication)

Authentification

  • C'est le processus de vérification de l'identité d'un utilisateur ou d'un système qui tente d'accéder à votre API
  • C'est comme demander à quelqu'un de prouver qui il est avant de lui permettre d'entrer dans un bâtiment sécurisé

Autorisation

  • Une fois l'identité vérifiée, l'autorisation détermine ce que l'utilisateur authentifié est autorisé à faire
  • C'est comme donner à quelqu'un un badge d'accès spécifique qui définit dans quels bureaux ou étages il peut entrer.

Symfony Security

Le composant Security de Symfony fournit des fonctionnalités de sécurité pour votre application (Web, API Rest)
